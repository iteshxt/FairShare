# Shared Expenses App

A transparent, auditable, and explainable shared expense management system with a staging CSV import engine that detects and handles data anomalies and membership timelines.

## Tech Stack
- **Frontend & Backend:** Next.js (App Router, TypeScript)
- **Styling:** Tailwind CSS & shadcn/ui
- **Database:** PostgreSQL (with Prisma ORM)
- **Authentication:** NextAuth.js
